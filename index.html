PhoenixGold Game Studios is an independent game studio focused on creating memorable family-friendly experiences with competitive gameplay, vibrant worlds, and lovable characters.

About Me

I'm Trevor Hicks, founder of PhoenixGold Game Studios and a graduate of Neumont College of Computer Science.

I specialize in game development, software engineering, and creating engaging experiences that blend technical excellence with creative design. My current flagship project is Eucalyptus Edge, a family-friendly weapon-based fighting game built in Unreal Engine 5.

Eucalyptus Edge

Cute Fighters. Serious Skills. Eucalyptus Edge is a family-friendly 3D weapon-based fighting game inspired by Soulcalibur. Explore the magical world of Verdantia where adorable Australian animal warriors battle against the mysterious Blight using unique weapons, strategic combat, and ring-out mechanics.

🎬 Watch the latest concept trailer above. This cinematic uses AI-assisted placeholder visuals to communicate the game's world and story while development continues in Unreal Engine 5.8.

Eucalyptus Edge Development Roadmap

Current progress toward the first playable prototype for PhoenixGold Game Studios' debut title.

Completed

Foundation & Branding

Game concept, studio branding, Kickstarter page, portfolio page, GitHub repository, and public project identity have been established.

Completed

World & Character Concepts

Verdantia, core arenas, fighter roster, PhoenixGold studio intro direction, and Eucalyptus Edge visual style have been designed.

In Progress

Unreal Engine 5.8 Setup

Project is now running in Unreal Engine 5.8 with imported character assets, consistent scaling, GitHub LFS support, and early project structure.

In Progress

Playable Character Pipeline

Multiple fighters have been imported and prepared with skeletons/animation compatibility. Remaining work includes cleanup, polish, and final rig testing.

In Progress

Main Menu Experience

Main menu direction is locked: cinematic animated background, streamlined AAA-style navigation, and a clear flow into character select and mode select.

Next Step

Combat Prototype

Build the first playable local versus prototype with movement, attacks, blocking, dodging, hit detection, camera behavior, UI feedback, and arena loading.

Upcoming

Training & AI Systems

Add training dummy logic, basic AI behavior, combat testing tools, and player feedback systems for health, Edge Energy, and match flow.

Future Phase

Expansion Toward Full Game

Expand roster, arenas, online multiplayer, story mode, cosmetics, supporter rewards, polish, optimization, and release preparation.
