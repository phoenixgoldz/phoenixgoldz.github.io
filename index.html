Studio Identity

A game studio built around original worlds.

PhoenixGold Game Studios is evolving from a student portfolio into a public-facing independent game studio brand. The focus is no longer personal résumé content, tech stacks, or unfinished side projects. The site now represents the studio, its creative direction, and its first commercial game.

The studio is focused on family-friendly games with strong visual identity, expressive gameplay, memorable characters, and worlds that can grow beyond a single release.

The first major PhoenixGold title is Eucalyptus Edge, a stylized 3D weapon-based arena fighting game planned for PC and Steam.

Studio Direction

Original Games. Memorable Worlds.

PhoenixGold is being shaped as a long-term studio brand with Eucalyptus Edge as the first published game and Verdantia as the first original universe.

Gameplay

Responsive Combat

Games should feel tight, readable, and rewarding. Eucalyptus Edge begins with skill-based dodging, blocking, parrying, attacks, and ring-out pressure.

Worldbuilding

Recognizable Worlds

Verdantia establishes the studio's first original IP world: colorful, dangerous, magical, and inspired by Australian eucalyptus forests.

Characters

Memorable Fighters

Every fighter is designed around silhouette, weapon identity, personality, and a consistent color language that supports a polished roster.
