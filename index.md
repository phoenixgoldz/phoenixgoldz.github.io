### 🎮 [Frostpaw Dash](https://pheonixgold.itch.io/frostpaw-dash)
A fantasy endless runner for Android where you guide a chibi white leopard through magical biomes like floating crystal caverns and Easter-themed levels.

**Version 1.4.08 Highlights**:
- Real-time leaderboard with arcade-style initials entry
- Async scene loading for <3s transitions
- Persistent vibration settings + performance boosts
- Built with Unity 6, C#, TextMeshPro, Post Processing

A candy-themed kart racer inspired by *Diddy Kong Racing*, built in Unreal Engine 5. Explore lollipop forests, unlock new biomes, and defeat bosses to save the Candy Kingdom.
