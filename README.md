## Featured Project

# Eucalyptus Edge

**Cute Fighters. Serious Skills.**

Eucalyptus Edge is a family-friendly 3D weapon-based arena fighting game currently in production using **Unreal Engine 5.8**.

Inspired by the expressive movement and skill-based combat of classic 3D fighters, Eucalyptus Edge combines serious fighting mechanics with a colorful fantasy world inspired by Australian wildlife and eucalyptus forests.

Core features include:

- ⚔️ Responsive weapon-based 3D combat
- 🛡️ Blocking, dodging, counters, and parries
- ⚡ Edge Energy special abilities
- 🏟️ Ring-out arena victories
- 🎮 Controller-first local versus gameplay
- 🌿 Stylized arenas across the world of Verdantia
- 🐨 Seven planned Australian animal champions
- 🌑 The mysterious corruption known as the Blight
- 🌐 Future online multiplayer expansion

The project-wide performance target is a stable **60 FPS**.

## Current Playable Focus

The active Phase 1 prototype is built around:

### Wren vs. Ripper

- **Wren the Red Kangaroo** — Combat Ready
- **Ripper the Tasmanian Devil** — Combat Ready
- **Arena:** Eucalyptus Summit
- **Engine:** Unreal Engine 5.8
- **Target:** Polished controller-ready gameplay showcase

Wren and Ripper are the first production-ready fighters being used to prove the complete combat loop, match flow, UI, camera, animation, and arena systems.

The first public gameplay showcase will combine polished Unreal Engine gameplay footage with the existing cinematic trailer direction rather than being presented as an Early Access release.

## Arena Production

Planned and concepted arenas include:

- Eucalyptus Summit
- Crystal Caverns
- Frostpine Ridge
- Bamboo Harbor
- Sunbaked Outback
- Edge Festival Colosseum
- Moonlit Rainforest

**Eucalyptus Summit** is the active Phase 1 combat arena and is being developed as the first fully playable environment.

Arena production targets include:

- Soulcalibur-style opponent-facing combat
- Optimized arena boundaries

- Niagara waterfalls and environmental effects
